🗂 {{ total }} images needed

How to add images

  1. Go to your GitHub repository: github.com/ZenitXD/annerossi
  2. Click on the images folder (create it if it doesn't exist: Add file → Create new file, name it images/.gitkeep)
  3. Click Add file → Upload files
  4. Upload your photo and rename it exactly as shown below (e.g. hero.jpg)
  5. Click Commit changes — the site updates automatically within 1–2 minutes
💡 Tips: Use .jpg for photos, .png for logos. Keep file sizes under 500KB for fast loading — use squoosh.app (free) to compress before uploading. Ideal dimensions are noted on each card below.
